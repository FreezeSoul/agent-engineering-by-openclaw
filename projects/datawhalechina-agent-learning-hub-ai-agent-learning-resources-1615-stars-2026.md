# datawhalechina/Agent-Learning-Hub：AI Agent 学习路线与资料库

> 与本文（Google DeepMind SIMA 2）形成互补：SIMA 2 是 DeepMind 前沿研究的展示（具身智能 + 3D 虚拟世界），Agent-Learning-Hub 是中文社区对 AI Agent 知识的系统整理——前沿研究需要知识基础支撑，这条学习路线覆盖了从理论到实践的完整路径。

---

## 一、项目定位：系统化的 Agent 学习路线

Agent-Learning-Hub 是 Datawhale 社区维护的 AI Agent 学习资料库，目标是：

> "AI Agent 学习路线与资料库收集"

项目整理了 AI Agent 领域的核心知识点、经典论文、开源项目和实践指南。

---

## 二、内容结构

### 2.1 学习路线

典型路线包括：
- **基础概念**：LLM Agent 的核心组件（规划、记忆、工具使用）
- **架构模式**：ReAct、CoT、AutoGPT 等经典范式
- **工程实践**：Prompt 工程、工具集成、多 Agent 协作
- **前沿进展**：论文阅读、代码实现、最新研究追踪

### 2.2 资源类型

| 类型 | 内容 |
|------|------|
| 论文 | 经典 Agent 论文的中文解读 |
| 代码 | 主流 Agent 框架的示例实现 |
| 教程 | 一步一步的实践指南 |
| 路线图 | 从入门到精通的学习路径 |

---

## 三、与 SIMA 2 的主题关联

SIMA 2 是前沿研究的产物，但它建立在一系列基础技术之上：

| 层次 | SIMA 2 | Agent-Learning-Hub |
|------|--------|-------------------|
| 上层 | 3D 具身智能 | 知识整理 |
| 中层 | Gemini 多模态 | LLM Agent 架构 |
| 底层 | 深度学习基础 | 机器学习基础 |

Agent-Learning-Hub 提供了理解 SIMA 2 所需的知识基础——没有 LLM、没有多模态感知、没有规划算法，SIMA 2 无法实现。

---

## 四、价值

- **系统化**：不是零散文章，而是完整的学习路线
- **社区驱动**：持续更新，吸收最新研究
- **中文友好**：降低了中文开发者的学习门槛
