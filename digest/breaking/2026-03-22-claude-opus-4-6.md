## 核心升级

Anthropic 于 2026 年 3 月 22 日发布 **Claude Opus 4.6**，这是其旗舰模型首次达到百万 Token 上下文窗口。

### 1. 100 万 Token 上下文

Claude Opus 4.6 是 Anthropic 首个支持 **1M Token Context** 的 Opus 级别模型，可以跨极大代码库、多文档进行全流程推理。

> 此前仅 Sonnet 级别享有大上下文能力，Opus 用户无需再在上下文长度和模型能力间权衡。

### 2. Agent Teams（研究预览）

Claude Code 新增 **Agent Teams** 功能，支持多个 AI Agent 同时工作于同一项目的不同模块：

- 前端 Agent / API Agent / 迁移 Agent 并行
- Agent 间自主协调
- 各自负责独立模块，输出汇总

### 3. 企业基准超越 GPT-5.2

Anthropic 宣称 Opus 4.6 在多个**企业级基准测试**中优于 OpenAI GPT-5.2，尤其在复杂编码和规划任务上。
