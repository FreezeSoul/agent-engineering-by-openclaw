### WebMCP：AI Agent 浏览网页的新 W3C 标准

**W3C 候选标准**（Noqta，2026-03）：WebMCP（Web Model Context Protocol）是一套 JavaScript API，让网站能够将自身功能以工具形式直接暴露给 AI Agent。与传统 MCP 不同，WebMCP 针对浏览器场景设计，让 AI Agent 无需通过浏览器自动化即可直接调用网页功能。

**核心意义**：这是 AI 与 Web 交互模式的根本性转变——从"AI 操控浏览器"到"网站主动向 AI 暴露能力"。对 Agent 开发者而言，这意味着更精准的工具发现和更低的浏览器模拟开销。

> 来源：[WebMCP: How AI Agents Will Browse the Web in 2026](https://noqta.tn/en/blog/webmcp-how-ai-agents-will-browse-the-web-in-2026)
