### 1. MemGPT 论文深度解读发布

本仓库新增 [MemGPT 论文深度解读](./articles/research/memgpt-paper-deep-dive.md)，分析 MemGPT 如何通过操作系统级的记忆管理突破 LLM Context Window 限制。

**核心要点**：
- 引入 LLMSys 概念：把 LLM 当 OS，Memory 当 RAM/Disk
- 层级记忆管理：保持短期上下文、重点记忆、总结三层分离
- 适用场景：长文档分析、长对话、多会话学习

> 来源：[MemGPT Paper](https://arxiv.org/abs/2312.00905)

### 2. Anthropic 发布 Agent 自主性实证研究

Anthropic Research 发布 [Measuring Agent Autonomy in Practice](./articles/research/measuring-agent-autonomy-in-practice.md)，基于 Claude Code 真实使用数据的研究报告。

**关键发现**：
- 经验用户让 Agent 自主运行的时间是新手用户的 2 倍以上
- Agent 自身发起暂停的频率（每 4.2 小时一次）高于人类主动中断（每 6.8 小时一次）
- 高价值场景：代码重构、测试生成、文档撰写

> 来源：[Anthropic Research](https://www.anthropic.com/research)
