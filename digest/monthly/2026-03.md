### Karpathy Autoresearch + 16 GPU 集群（3/20）

**实验**：给 Claude Code 提供 16 GPU K8s 集群做 autoresearch

**结果**：
- 8 小时内提交 ~910 次实验
- 并行度改变搜索策略：单 GPU 只能 greedy hill-climbing，16 GPU 做 factorial grids（每波 10-13 个实验）
- **关键发现**：并行搜索能捕获 sequential search 错过的参数交互效应

**工程启示**：Agent 的计算规模（compute scaling）可能成为新研究维度。

> 来源: [SkyPilot Blog](https://blog.skypilot.co/scaling-autoresearch/)
