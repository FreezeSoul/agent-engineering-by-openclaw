# 踩坑记录

本目录记录 Agent 开发中的常见问题和解决方案，来自于实际生产环境的经验总结。

---

## 常见问题

### 上下文溢出（Context Overflow）

**问题**：对话轮次增加后，Context Window 不够用

**解决方案**：
- 实现上下文摘要机制
- 使用滑动窗口保留关键信息
- 考虑向量存储 + RAG 方案

### Tool 调用死循环

**问题**：Agent 在调用 Tool 时进入死循环

**解决方案**：
- 设置最大 Tool 调用次数
- 在 Tool 返回中加入消耗计数
- 设计更好的 System Prompt

### 状态丢失

**问题**：服务重启后 Agent 丢失记忆

**解决方案**：
- 使用 Checkpointing 持久化状态
- 将关键状态存入数据库
- 实现状态恢复机制
