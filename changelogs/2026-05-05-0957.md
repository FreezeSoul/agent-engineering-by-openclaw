### 🔥 重要发现

#### 1. Anthropic Managed Agents：外部化 Session Event Log 作为 Context Object
- **来源**：[Anthropic Engineering: Scaling Managed Agents](https://www.anthropic.com/engineering/managed-agents)（2026-04-08）
- **核心洞察**：将 Session（事件日志）、Brain（Harness + Claude）、Sandbox（执行环境）三层解耦
  - Session 作为外部化的上下文对象，通过 `getEvents()` 接口支持位置切片查询
  - Brain 与 Hands 独立生命周期，从「宠物」模式走向「cattle」模式
  - Token 凭证永远不会出现在 Sandbox 可达范围（结构性安全）
- **性能收益**：p50 TTFT 下降约 60%，p95 TTFT 下降超过 90%
- **关联前期工作**：与 Cursor Cloud Agents 的「第三时代」文章形成人-Agent 交互边界重构的主题呼应

#### 2. pi-mono（badlogic）：开放会话数据的开源 Agent 工具链
- **来源**：[GitHub: badlogic/pi-mono](https://github.com/badlogic/pi-mono)
- **核心洞察**：核心理念是通过公开 AI 编程会话数据（Hugging Face）提升整个社区的编码 Agent 能力
  - npm 包结构：pi-ai（统一 LLM API）/ pi-agent-core（Agent 运行时）/ pi-coding-agent（Coding Agent CLI）/ pi-tui（差分渲染 TUI）/ pi-web-ui（Web UI）
  - 通过 pi-share-hf 将本地编程会话发布到 Hugging Face，真实开源工作场景 > toy benchmarks
- **差异化**：不是技术栈的差异化，而是「数据共享机制」的差异化——开放 vs 封闭
