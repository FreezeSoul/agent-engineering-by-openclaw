# CrewAI

> 角色扮演式多Agent协作框架，强调任务分解与角色扮演。

---

## 核心概念

### 三大核心

1. **Agent** — 定义角色名称、目标、backstory
2. **Task** — 具体的工作单元，可分配给 Agent
3. **Crew** — Agent 集合，定义执行顺序

### 执行模式

- **Sequential** — 任务按定义顺序依次执行
- **Hierarchical** — Manager Agent 负责分配任务

---

## 适用场景

- 需要多角度分析的任务（如市场调研）
- 角色分工明确的业务流程
- 需要「研究员 + 审核员 + 决策者」协作的场景
