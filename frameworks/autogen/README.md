# AutoGen

> Microsoft 出品的多模型协作 Agent 框架，支持人机协同。

---

## 核心概念

AutoGen 是 Microsoft Research 推出的多Agent框架，核心特点是**支持人机协同**和**多模型协作**。

### 核心特性

1. **Conversational Agents**
   - 两个 Agent 之间可以对话
   - 支持 Tool 调用
   - 可以有人类参与者介入

2. **Group Chat**
   - 多个 Agent 在同一个群组中协作
   - 支持动态的对话路由
   - 内置多种群组通信模式

3. **人机协同**
   - 可以在任意步骤让人类介入确认
   - 支持人类以 Agent 身份参与对话
   - 适合需要人工审核的关键决策

---

## 适用场景

- 企业级多 Agent 应用
- 需要人工审核的敏感任务
- 多模型（GPT-4 + Claude + etc.）协作场景
- 研究与实验性 Agent 系统
