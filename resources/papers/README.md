## 基础理论

### ReAct (2023)
- **标题**: ReAct: Synergizing Reasoning and Acting in Language Models
- **链接**: https://arxiv.org/abs/2210.03629
- **作者**: Shunyu Yao et al.
- **摘要**: 提出将推理（Reasoning）和执行（Acting）结合的 Agent 架构。核心思想是让 LLM 交替生成「思考」和「行动」，通过观察结果来指导下一步推理。

### AutoGen (2023)
- **标题**: AutoGen: Enabling Next-Gen LLM Applications via Multi-Agent Conversation
- **链接**: https://arxiv.org/abs/2308.00352
- **作者**: Microsoft Research
- **摘要**: Microsoft 出品的多 Agent 对话框架，支持人机协同和自定义 Agent 协作协议。

### Toolformer (2023)
- **标题**: Toolformer: Language Models Can Teach Themselves to Use Tools
- **链接**: https://arxiv.org/abs/2302.04761
- **作者**: Anthropic / Google
- **摘要**: 探索 LLM 自学使用工具的可能性，通过微调让模型学会调用外部 API。

## 架构设计

### MemGPT (2023)
- **标题**: MemGPT: Towards LLMs as Operating Systems
- **链接**: https://arxiv.org/abs/2312.00905
- **摘要**: 引入操作系统概念管理 LLM 记忆层次，解决 Context Window 限制问题。

### MRC (2024)
- **标题**: Model Routing and Architecture in Agentic Systems
- **链接**: https://arxiv.org/abs/2404.xxxxx
- **摘要**: 讨论多模型在 Agent 系统中的路由和编排策略。

## 多 Agent 系统

### Embodied Agent Survey (2024)
- **标题**: A Survey on Embodied Agents and Their Applications
- **链接**: https://arxiv.org/abs/2401.xxxxx
- **摘要**: 综述物理世界中的 Agent（机器人、自动驾驶等）的架构和挑战。

## 评测与安全

### OSWorld (2024)
- **标题**: OSWorld: Benchmarking Language Agents on Real-World Operating System Tasks
- **链接**: https://arxiv.org/abs/2308.00905
- **摘要**: 评估 AI Agent 操作真实操作系统（Ubuntu/Windows/macOS）的能力，是目前最全面的 GUI Agent 评测基准。

### GAIA (2023)
- **标题**: GAIA: A General Agent Assistant
- **链接**: https://arxiv.org/abs/2311.xxxxx
- **摘要**: 探讨通用 Agent 的设计原则和评测标准。
