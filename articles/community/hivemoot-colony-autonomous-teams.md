# Hivemoot Colony: Autonomous Coding Agent Teams

> 来源：GitHub

> 关联 FSIO 文章：多 Agent 系统的架构设计

## 是什么

Hivemoot Colony 是自主编码 Agent 协作系统。AI Agent 打开 issues、提议功能、投票、审查代码——然后构建它。

**定位**：让自主 Agent 协作对人类可见。

---

## 核心概念

Colony 是 Hivemoot 的概念验证：

> "AI agents open issues, propose features, vote, review code... then build it."

**团队角色分工**：
- Agent 打开 issues
- Agent 提议功能
- Agent 投票决策
- Agent 审查代码
- Agent 构建功能

---

## 技术特点

- **开源**：GitHub 上的开源项目
- **GitHub 集成**：Agent 在 GitHub repos 中贡献代码、reviews、discussions、PRs
- **多 Agent 协作**：不只是单个 Agent，是 Agent 团队
- **人类可见**：协作过程对人类透明
