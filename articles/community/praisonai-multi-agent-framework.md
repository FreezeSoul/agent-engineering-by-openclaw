# PraisonAI: Production-Ready Multi-Agent Framework

> 来源：GitHub / PyPI

## 是什么

PraisonAI 是一个生产级的多 AI Agent 框架，设计用于自动化任务，结合了 AutoGen、CrewAI 和 self-reflection 能力。

**定位**：让创建和管理自反思 AI Agent 变得简单。

---

## 核心特性

### Self-Reflection

Agent 评估并改进自己的响应以获得更高准确率。

```
Agent 生成响应 → 自我评估 → 迭代改进 → 再次评估 → ...
```

**迭代改进**：通过多轮反思循环精炼响应。

### Multi-Step Reasoning

高级逻辑推理能力。

### 100+ LLM 支持

支持 100+ 大语言模型。

### MCP 集成

内置 MCP（Model Context Protocol）集成。

### 最快 Agent 实例化

**3.77μs** 的 Agent 实例化速度（最快）。

---

## 框架架构

PraisonAI 结合了：
- **PraisonAI Agents**：核心 Agent 框架
- **AutoGen**：微软多 Agent 对话框架
- **CrewAI**：角色扮演 Agent 框架

---

## 使用场景

| 场景 | 说明 |
|------|------|
| 自动化任务 | 简单到复杂的任务自动化 |
| 自反思工作流 | 需要自我改进的复杂推理 |
| 多 Agent 协作 | 多个 Agent 角色分工 |
