## 三、Memory-Skill Contract 1st-party 标准化窗口（R670 NEW 触发）

### 3.1 Memory-Skill Contract 定义

**Layer 4 ↔ Layer 3 协议契约**：

```
Layer 4 (hindsight Memory) ↔ Layer 3 (Skill Registry):

Memory context → Skill activation:
  - memory fact → skill trigger condition
  - memory context → skill execution environment
  - memory update → skill version bump

Skill activation → Memory update:
  - skill execution result → memory fact
  - skill completion → memory state update
  - skill error → memory error log
```
